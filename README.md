# 🍄 The Last of Us — Fan Site

Fan site dedicado à franquia **The Last of Us**, desenvolvido com HTML, CSS e JavaScript puro — sem frameworks ou dependências externas além de fontes e ícones.

---

## 🎮 Sobre o Projeto

Site de fã cobrindo os dois jogos principais da franquia da Naughty Dog, com foco em design imersivo e conteúdo detalhado sobre o universo de TLOU.

---

## ✨ Funcionalidades

| Recurso | Descrição |
|---|---|
| 🔫 Cursor personalizado | Mira animada que substitui o cursor padrão (desktop) |
| ⏳ Loading screen | Tela de carregamento temática com barra de progresso |
| 🌅 Modo sépia | Alternância entre tema escuro e sépia, salvo no localStorage |
| 🍂 Esporos flutuantes | Partículas animadas de fundo geradas via JavaScript |
| 🎠 Carrosséis | Carrossel de personagens e infectados com suporte a swipe e teclado |
| 🗺️ Mapa interativo | Mapa dos EUA com os locais dos dois jogos e tooltips informativos |
| 🖼️ Galeria de fãs | Upload de fotos via drag & drop com lightbox e armazenamento local |
| ❓ Quiz | "Qual personagem de TLOU você seria?" com 5 resultados possíveis |
| 📱 Menu lateral | Painel de navegação com seções extras |
| ♿ Acessibilidade | ARIA labels, navegação por teclado e suporte a swipe em todo o site |

---

## 📂 Estrutura

```
/
├── index.html        # Estrutura e conteúdo do site
├── style.css         # Estilos, animações e responsividade
├── script.js         # Toda a interatividade
└── images/           # Assets visuais (webp, jpg, svg)
```

---

## 🛠️ Tecnologias

- **HTML5** — estrutura semântica
- **CSS3** — animações, variáveis CSS, grid e flexbox
- **JavaScript** — vanilla, sem frameworks
- **[Bebas Neue / Crimson Text / Oswald](https://fonts.google.com)** — tipografia via Google Fonts
- **[Font Awesome 6](https://fontawesome.com)** — ícones

---

## 📖 Seções do Site

1. **Hero** — imagem de abertura com animação de scroll
2. **Em Números** — estatísticas das duas franquias
3. **Sobre os Jogos** — tabs com ficha técnica do Part I e Part II
4. **Comparativo** — Part I vs Part II frente a frente
5. **Personagens** — carrossel com todos os personagens principais
6. **Infectados** — carrossel com os estágios da infecção
7. **Facções** — FEDRA, Vaga-lumes, WLF, Seraphitas e mais
8. **Jornada** — timeline dos eventos dos dois jogos
9. **Trilha Sonora** — Gustavo Santaolalla e Mac Quayle
10. **Mapa da Jornada** — locais visitados nos dois jogos
11. **A Série** — jogos, DLCs e série HBO
12. **Easter Eggs** — segredos escondidos pela Naughty Dog
13. **Galeria de Fãs** — upload e visualização de fotos
14. **Quiz** — descubra qual personagem você seria

---

## 🚀 Como Rodar

Basta abrir o `index.html` diretamente no navegador — não precisa de servidor ou instalação.

```
Abra o index.html no Chrome, Firefox ou Edge
```

> Para melhor experiência, use um servidor local (ex: extensão Live Server no VS Code).
